### Propose

### Vagrant Environment Setup

If you don't have virtual box download it first:
https://www.virtualbox.org/

Instruction to run the project:
  * cd vm
  * vagrant up
  * vagrant ssh
  * cd ~/propose
  * python manage.py runserver 0.0.0.0:8000
